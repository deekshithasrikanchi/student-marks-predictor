# Student Marks Predictor

This project predicts student marks based on study hours using a simple linear regression model.

## Overview
The model learns the relationship between study hours and marks and provides predictions based on user input. A basic visualization is also included.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
python marks_predictor.py

## Output
- Accepts study hours as input
- Predicts marks
- Displays a graph
